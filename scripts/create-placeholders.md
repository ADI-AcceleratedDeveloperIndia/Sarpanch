# Creating Placeholder Images

Since actual village photos are not included, you can create simple placeholder images or use online tools.

## Option 1: Use Online Placeholder Services

Visit these URLs and save the images to `public/placeholders/`:

- `sarpanch1.png`: https://via.placeholder.com/400x400/FFA500/FFFFFF?text=Sarpanch
- `village1.png`: https://via.placeholder.com/400x400/8B7355/FFFFFF?text=Village
- `leader1.png`: https://via.placeholder.com/400x400/FFA500/FFFFFF?text=Leader
- `work1.png`: https://via.placeholder.com/800x600/8B7355/FFFFFF?text=Road+Work
- `work2.png`: https://via.placeholder.com/800x600/8B7355/FFFFFF?text=Borewell
- `photo1.png`: https://via.placeholder.com/600x400/8B7355/FFFFFF?text=Gallery+Photo+1
- `photo2.png`: https://via.placeholder.com/600x400/8B7355/FFFFFF?text=Gallery+Photo+2

## Option 2: Use Real Photos

Replace the placeholder files with actual village photos:
- Recommended size: 800x600px for gallery/work photos
- Recommended size: 400x400px for profile photos
- Format: PNG or JPG
- Optimize images before uploading for better performance
